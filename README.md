# Nombre del Proyecto: Recuperatorio de Ing Software -

**Descripción:** Landing page desarrollada como parte del recuperatorio de la materia, integrando Git y Docker.

**Tecnologías utilizadas:**
HTML5 y CSS3
GitHub
Docker

**Pasos para ejecutar la aplicación:**
1. Tener Docker instalado y corriendo.
2. Descargar y ejecutar la imagen desde Docker Hub con:
 "docker run -d -p 8080:80 65487321/recuperatorio-infra:1.0"
3. Acceder a "http://localhost:8080"

**Comandos utilizados para construir la imagen:**
"docker build -t 65487321/recuperatorio-infra:1.0 ."
